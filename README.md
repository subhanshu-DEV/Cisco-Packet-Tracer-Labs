# Cisco Packet Tracer Beginner Labs

This repository contains my networking practice labs created using Cisco Packet Tracer.

## Tools Used

* Cisco Packet Tracer

## Labs Completed

* Basic LAN Topology
* Bus Topology
* Ring Topology
* Star Topology
* Mesh Topology
* Routing Between Two Networks

## Topics Practiced

* Network Topologies
* Connecting PCs, Switches, and Routers
* IPv4 Address Configuration
* Router Interface Configuration
* Inter-Network Routing
* Connectivity Testing using Ping
* Basic Network Troubleshooting

## Lab Files

* Basic Lan Topology.pkt
* bus-topology-lab.pkt
* ring-topology.pkt
* star-topology-lab.pkt
* mesh-topology-lab.pkt
* routing-between-two-networks.pkt

## Learning Goals

* Understand computer networking fundamentals
* Learn different network topologies
* Configure routers and switches
* Practice IP addressing and subnetting basics
* Build and test small network infrastructures
